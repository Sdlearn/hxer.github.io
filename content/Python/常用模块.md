## sys

```
executable: 返回python解释器的路径, e.g. '/usr/bin/python'
platform: 返回操作系统平台, e.g. 'linux2'
path: 模块搜索路径，Python 在导入期间将在其中的目录列表中寻找模块。最前面的空字符串 '' 是指当前目录
```
